How it started

IDEA About DR Search

The purpose of the project entitled as “DR. SEARCH” is to computerize the enquiry of various medical services to develop software which is user friendly simple, fast, and cost effective. It deals with the collection of Doctor’s information, Diagnosis details, blood banks and Hospitals details etc. Traditionally, it was done manually. The main function of the system is to provide patients a flexible system for hospital details, Blood bank details etc. and retrieve these details as and when required, and also to manipulate these details meaningfully System input contains doctor details, diagnosis details, while system output is to get these details on to the screen.

The project “DR. SEARCH” can be entered without any password systems for enquiry and appointment purposes, to manipulate the server data using a username and password. It is accessible by an administrator. Only he/she can add data into the database. The data can be retrieved easily. The data are well protected for personal use and makes the data processing very fast.
